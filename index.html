    🎯 Objectifs du Cours¶

    Cette documentation regroupe l'ensemble des supports de cours et exercices pour le module de mathématiques en A2. Elle couvre trois grands domaines :

    1. Analyse Multivariable (ETS15)
    2. Fonctions de plusieurs variables
    3. Intégration multiple

    4. Applications pratiques

    5. Physique Mathématique (ETS14)

    6. Oscillateurs harmoniques
    7. Ondes mécaniques

    8. Thermodynamique

    9. Calcul Vectoriel (ETS16)

    10. Opérateurs différentiels
    11. Applications géométriques
    12. Champs vectoriels

    💡 Comment Utiliser cette Documentation¶

    1. Commencez par consulter le Planning de Révision
    2. Étudiez les fiches théoriques de chaque module
    3. Pratiquez avec les exercices correspondants
    4. Utilisez la Synthèse Finale pour les révisions
    5. Testez vos connaissances avec le Test Blanc

    🎯 Objectif¶

    Ce site regroupe l'ensemble des supports de révision pour les examens du semestre 3, couvrant les modules ETS14, ETS15 et ETS16.
